1Project Purpose

GymTrack is a comprehensive database management system built on REST API architecture with role-based authorization, enabling multi-branch gym chains to manage member registrations, subscription packages, trainer assignments, class schedules, equipment maintenance, and financial reporting through a single centralized web platform. Administrators track branch performance, trainers manage class schedules, and members monitor their subscription status in real time.

7Security Measures

🔐
Bcrypt Password Hashing
Passwords are stored as one-way hashes. Even in a database breach, original passwords remain hidden. Each hash includes a random salt.
🛡️
Prepared Statements
All SQL queries are parameterized. Full protection against SQL Injection attacks. User input is never interpreted as SQL code.
🚪
Auth Guard
Every API endpoint passes through authentication checks. Unauthorized requests are rejected. Session-based login management.
🧹
XSS Protection
User input is HTML-escaped before render. Script injection is blocked. A helper escapeHtml() function is used throughout.
👮
Role-Based Access
Admin, Trainer, and Member roles hold distinct permissions. Each user only accesses resources they are authorized for.
✅
Data Validation
Email format checks, required field validation, and foreign key constraints preserve data integrity at every layer.
